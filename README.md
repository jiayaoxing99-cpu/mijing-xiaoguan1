# 谜境小馆 - 猜谜游戏

## 项目简介

「谜境小馆」是一款基于原生 HTML、CSS、JavaScript 开发的网页猜谜游戏。游戏采用国风小馆风格设计，提供简单、中等、困难三种难度以及混合挑战模式，内置 30 道精选谜语，涵盖字谜、动物谜、物品谜、自然谜等多种类型。

玩家可以体验完整的游戏流程：从首页开始，选择难度，进入答题，使用提示或跳过，最终查看通关结果并分享成绩。

## 运行方式

本项目无需构建，无需依赖后端或外部 CDN，直接双击 `index.html` 即可运行。

### 环境要求
- 现代浏览器（Chrome、Firefox、Safari、Edge 等）
- 无需服务器支持

### 启动步骤
1. 下载或克隆项目文件
2. 打开 `mijing-xiaoguan` 文件夹
3. 双击 `index.html` 文件
4. 在浏览器中开始游戏

## 核心功能

### 🎮 游戏玩法
- **难度选择**：简单（10题）、中等（10题）、困难（10题）、混合挑战（随机10题）
- **答题系统**：输入答案后提交，支持回车键快捷提交
- **答案验证**：支持别名匹配，忽略大小写和前后空格
- **提示功能**：每题最多可查看 2 条提示，每条提示扣 2 分
- **跳过功能**：可跳过当前题目，不得分并显示正确答案

### 📊 成绩系统
- **得分规则**：答对得 10 分，使用提示扣 2 分/次
- **历史最高分**：使用 localStorage 保存历史最高分
- **通关称号**：根据成绩获得不同称号（谜语大师、猜谜高手、解谜达人等）
- **用时统计**：记录完成游戏的时间

### 📱 用户体验
- **响应式设计**：适配手机和电脑屏幕
- **动画效果**：答对绿色反馈、答错红色反馈、卡片抖动动画
- **答案解析**：答对或跳过显示详细答案解释
- **分享功能**：一键复制成绩分享文案
